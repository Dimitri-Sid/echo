8 Interpretation Guide

How to read the results and understand the proxy's limitations.

The TRBV usage diversity proxy is analogous to the NanoString nCounter TCR Diversity score. It quantifies V-gene breadth and evenness. It cannot distinguish CDR3-level clonal expansion from V-gene usage bias driven by germline or developmental constraints.
✓ High ρ (>0.7), points near 1:1 line
The TRBV proxy is a reasonable diversity surrogate in this context — typical of sorted CD8 T cells with no strong V-gene bias.
△ Low ρ or systematic offset
Proxy breaks down; full TCR sequencing (CDR3 level) is required for reliable clonality estimation in this context.
✗ High HTRBV, Low Hclonotype
Multiple TRBV genes used, but each harbours a single dominant clone. The proxy overestimates diversity. Classic: vaccine-driven oligoclonal expansion using diverse TRBV families.
✗ Low HTRBV, High Hclonotype
Few TRBV genes used, but each harbours many distinct CDR3s. The proxy underestimates diversity. Classic: germline V-gene bias with no actual clonal expansion.
✗ Red bars in failure-mode plot
A single clonotype accounts for >50% of cells within a TRBV family — the direct failure case where CDR3-level resolution is essential.

How to reproduce

Run the following in R from the repository root to re-generate all analysis outputs and re-render this report:

# Install dependencies (once)
BiocManager::install("zellkonverter")
install.packages(c("rmarkdown","DT","kableExtra","plotly",
                   "here","dplyr","tidyr","purrr","ggplot2",
                   "ggrepel","scales","data.table",
                   "RColorBrewer","patchwork","pheatmap"))

# Render HTML report
rmarkdown::render("analysis/validation/tcr_diversity_report.Rmd",
                  output_file = "analysis/validation/report.html")

# Or run the plain analysis script (no HTML output)
source("analysis/validation/run_validation_analysis.R")
