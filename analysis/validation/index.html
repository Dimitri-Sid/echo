6 CD4 vs CD8 Stratification

The NanoString TCR Diversity assay is designed for defined T cell compartments, not bulk T cells. The key validation question is: within CD8 T cells, does the TRBV proxy track CD8-specific clonotype diversity? Each point = one sample within one subset.

7 Pseudoclonotype Recapitulation

Cells sharing the same gene-combo string (e.g., TRAV:TRAJ:TRBV:TRBJ) are assigned the same pseudoclonotype label. The analysis measures how faithfully each combo level recapitulates true CDR3-based clonotypes — stratified by CD4 and CD8.

Purity: of cells sharing a pseudoclonotype, what fraction share the dominant real clonotype? (high = good) · Collision: fraction of pseudoclonotypes containing >1 real clonotype (low = good) · Split: fraction of real clonotypes spanning >1 pseudoclonotype (low = good)

8 RNA-seq vs TCR-seq Gene Usage

For each cell, the dominant expressed TRBV/TRAV gene in the scRNA-seq matrix is compared to the VDJ-sequencing-assigned gene call. Concordance = fraction of cells where both agree. Datasets without a matched expression matrix show no data here.

11 Interpretation Guide

How to read the results and understand each proxy's limitations.

The TRBV proxy measures V-gene breadth/evenness — analogous to the NanoString nCounter TCR Diversity score. None of the gene-segment proxies can resolve CDR3-level clonotypes; the full four-segment paired metric (TRAV:TRAJ:TRBV:TRBJ) narrows the gap the most without full CDR3 sequencing.
✓ High ρ (>0.7)
Proxy is a reasonable diversity surrogate in this context.
△ Low ρ or offset from 1:1
Proxy breaks down; full CDR3 TCR sequencing required.
✗ High H_TRBV, Low H_clonotype
Multiple TRBV genes used, but each harbours a single dominant clone (proxy overestimates diversity).
✗ Low H_TRBV, High H_clonotype
Few TRBV genes used, but each harbours many distinct CDR3s (germline V-gene bias without clonal expansion).
✗ Red bars in failure-mode plot
Single clone >50% of cells within a TRBV family — direct failure case.
