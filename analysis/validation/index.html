11 Interpretation Guide

How to read the results and understand each proxy's limitations.

The TRBV proxy measures V-gene breadth/evenness — analogous to the NanoString nCounter TCR Diversity score. None of the gene-segment proxies can resolve CDR3-level clonotypes; the full four-segment paired metric (TRAV:TRAJ:TRBV:TRBJ) narrows the gap the most without full CDR3 sequencing.
✓ High ρ (>0.7)
Proxy is a reasonable diversity surrogate in this context.
△ Low ρ or offset from 1:1
Proxy breaks down; full CDR3 TCR sequencing required.
✗ High H_TRBV, Low H_clonotype
Multiple TRBV genes used, but each harbours a single dominant clone (proxy overestimates diversity).
✗ Low H_TRBV, High H_clonotype
Few TRBV genes used, but each harbours many distinct CDR3s (germline V-gene bias without clonal expansion).
✗ Red bars in failure-mode plot
Single clone >50% of cells within a TRBV family — direct failure case.
